# 기관 지침서 초안 규격

이 문서는 향후 각 기관 또는 서브 에이전트의 실제 프롬프트/지침서를 만들 때 따를 공통 규격이다.

## 각 기관 문서에 반드시 들어갈 항목
1. 기관명
2. 기관의 존재 목적
3. 맡는 일
4. 맡지 않는 일
5. 입력 형식
6. 출력 형식
7. 다른 기관과의 인수인계 방식
8. 품질 기준
9. 금지사항
10. 영의정에게 보고하는 형식

## 예시 적용 대상
- 승정원 지침서
- 집현전 지침서
- 병조 지침서
- 도화서 지침서
- 사헌부 지침서

## 원칙
- 기관은 멋보다 기능이 먼저다.
- 프롬프트는 길이보다 경계가 명확해야 한다.
- 입력/출력이 अस्पष्ट하면 독립 기관으로 세우지 않는다.
- 영의정은 각 기관 지침서를 통합 관리한다.
