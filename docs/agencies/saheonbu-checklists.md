# 사헌부 검수 체크리스트

## 1. 공통 체크리스트
- 왕명이 제대로 반영되었는가
- 확인되지 않은 내용을 사실처럼 단정했는가
- 앞뒤 논리가 맞는가
- 불필요한 중복이 있는가
- 결과물이 실제로 쓸 만한 수준인가
- 빠진 핵심 항목이 있는가

## 2. 기획안 검수
- 문제 정의가 명확한가
- 목표와 수단이 연결되는가
- 범위가 과도하게 퍼지지 않았는가
- 우선순위가 타당한가

## 3. 개발 결과 검수
- 실제 변경 내용이 분명한가
- 실행/테스트 여부가 밝혀졌는가
- 안 된 것을 된 것처럼 쓰지 않았는가
- 남은 위험과 한계가 드러나는가

## 4. 시각/디자인 검수
- 보기 좋음보다 목적 적합성이 확보되었는가
- 구현 가능한 수준으로 설명되었는가
- 사용자 흐름이 이해되는가

## 5. 문서/보고 검수
- 구조가 자연스러운가
- 결론과 근거가 이어지는가
- 그럴듯한 말로 빈 곳을 덮지 않았는가
