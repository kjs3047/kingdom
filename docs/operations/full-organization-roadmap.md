# 전체 조직 확장 및 누락 점검 로드맵

## 목적
초기 기획에서 제시된 전체 왕국 구조를 다시 점검하여, 아직 실전 문서화되지 않은 기관과 기능을 식별하고 확장 순서를 정한다.

## 1. 이미 실전 정의가 진행된 핵심 기관
- 승정원
- 집현전
- 병조
- 도화서
- 사헌부
- 영의정

## 2. 아직 실전 문서화가 미진한 기관
### 한성부
- 운영지원
- 고객응대 및 문의 처리
- 데이터 정리 및 운영 질서 유지

### 이조
- 에이전트 역할 배치
- 권한 등급 관리
- 승격/해임 기준
- 성과 기준 관리

### 호조
- 토큰/API 비용 관리
- 예산 배분
- 사용량 집계
- 결제/수익모델 검토

### 예조
- 콘텐츠
- 브랜딩 메시지
- 교육
- 홍보
- 세일즈
- 파트너십

### 형조
- 약관
- 저작권
- 개인정보
- 윤리 기준
- 위험 판정

### 공조
- 시스템 구조 설계
- 내부 툴 제작
- 실험 환경
- 기반 기술 관리

## 3. 초기 설계 대비 현재 누락 또는 약한 부분
- 한성부의 실전 지침 부재
- 6조의 개별 실전 지침 부재
- 비용/정책/법무/운영지원 흐름 문서 부족
- 이미지/영상 생성 실무 흐름의 상세화 부족
- 장기 확장 기관별 도구 체계 미확정

## 4. 확장 우선순위
1. 예조
2. 공조
3. 한성부
4. 형조
5. 호조
6. 이조

## 5. 원칙
- 처음 설계의 전체 조직은 유지하되, 실전 문서화는 단계적으로 진행한다.
- 핵심기관이 시험 운용에 들어간 뒤, 반복 필요가 생기는 기관부터 확장한다.
- 이름만 있는 기관을 늘리지 않고, 역할과 도구가 함께 정해질 때 실전화한다.
