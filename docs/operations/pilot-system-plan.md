# 나랏말 AI 왕국 초안 시스템 구축 계획

## 목표
문서상 왕국을 실제로 돌릴 수 있는 최소 운영 시스템으로 전환한다.

## 1단계: 핵심 기관 지침 완성
- 승정원
- 집현전
- 병조
- 도화서
- 사헌부

완료 기준:
- 역할, 입력, 출력, 금지사항, 보고 형식이 문서화될 것

## 2단계: 도구 체계 정리
- 기관별 핵심 도구 정의
- 실무 가능한 수준의 작업대 확정
- 병조를 중심으로 실행 가능한 구조 우선 구축

## 3단계: 파일/문서 구조 정리
- docs/agencies
- docs/operations
- docs/governance
- docs/templates
형태로 점차 정리

## 4단계: 시험 왕명 시나리오 작성
예:
- 소개 웹페이지 초안 제작
- 서비스 기획안 작성
- 발표자료 개요 작성
- 마케팅 문구 초안 작성
- 결과물 검수 요청

## 5단계: 실전 테스트
- 왕명이 들어오면 영의정이 해석
- 필요 시 기관 역할에 따라 분기
- 결과를 취합
- 사헌부 검수
- 왕에게 보고

## 초기 성공 기준
- 간단한 홈페이지 제작 또는 문서형 프로젝트를 한 번 끝까지 수행할 수 있을 것
- 영의정이 분기와 취합을 일관되게 해낼 것
- 사헌부가 최소한의 검수 역할을 수행할 것
