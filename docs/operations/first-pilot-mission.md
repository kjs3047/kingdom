# 첫 시험 임무 제안

## 임무명
나랏말 AI 왕국 소개용 단일 페이지 초안 제작

## 왜 이 임무가 좋은가
- 집현전, 도화서, 병조, 사헌부를 모두 가볍게 시험할 수 있다.
- 결과물이 눈에 보여 평가가 쉽다.
- 지나치게 큰 프로젝트가 아니라 초안 시스템 검증에 적합하다.

## 권장 흐름
1. 집현전: 페이지 목적, 대상, 핵심 메시지 정리
2. 도화서: 섹션 구조와 시각 톤 제안
3. 병조: 실제 HTML/CSS 또는 간단한 웹 초안 구현
4. 사헌부: 과장, 구조 허점, 미완성 요소 검수
5. 영의정: 최종 보고

## 성공 기준
- 실제 파일이 생성될 것
- 왕국 소개 메시지가 분명할 것
- 최소한의 시각 구조가 있을 것
- 검수 의견이 붙을 것
