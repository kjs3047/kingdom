# 시험 왕명 시나리오

## 시나리오 1: 소개용 홈페이지 제작
- 왕명: 나랏말 AI 왕국 소개용 간단한 홈페이지를 만들어라.
- 기본 흐름: 집현전(구조) -> 도화서(화면/표현) -> 병조(구현) -> 사헌부(검수) -> 영의정(보고)

## 시나리오 2: 서비스 기획안 작성
- 왕명: 새 서비스의 기획안을 작성하라.
- 기본 흐름: 집현전 -> 사헌부 -> 영의정

## 시나리오 3: 발표자료 개요 작성
- 왕명: 투자자 대상 발표자료 개요를 짜라.
- 기본 흐름: 집현전 -> 도화서 -> 사헌부 -> 영의정

## 시나리오 4: 기능 수정 요청
- 왕명: 기존 코드의 로그인 흐름을 고쳐라.
- 기본 흐름: 병조 -> 사헌부(필요 시) -> 영의정

## 시나리오 5: 홍보 문구 작성
- 왕명: 제품 홍보 문구를 작성하라.
- 기본 흐름: 집현전 -> 도화서(브랜드 톤 보조 가능) -> 사헌부(필요 시) -> 영의정
