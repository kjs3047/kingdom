# 기관별 산출물 규격

## 목적
기관이 실제로 일했다는 흔적을 일관된 형식으로 남기기 위해, 기관별 최소 산출물 규격을 정한다.

## 승정원
- 작업 보드 갱신
- 상태 변경 기록
- 병목 보고 갱신

## 집현전
- 목표 정의
- 핵심 메시지
- 구조 초안

## 도화서
- 시각 방향
- 톤앤매너
- 연출 포인트
- 보정 포인트

## 병조
- 변경 파일 목록
- 구현 내용
- 확인 결과
- 남은 한계

## 사헌부
- 판정
- 핵심 문제
- 수정 우선순위
- 재검수 필요 여부

## 원칙
- 모든 기관은 최소 하나의 실제 산출 파일을 남긴다.
- 산출물은 영의정이 아니라 해당 기관 이름으로 구분 가능해야 한다.
- 산출물이 없으면 그 기관은 실제로 일한 것으로 보지 않는다.
