# 작업 보드 - 나랏말 AI 왕국 소개 랜딩페이지

## 작업명
나랏말 AI 왕국 소개용 랜딩페이지 제작

## 왕명 요약
히어로 섹션에 강한 애니메이션을 넣고, 멋진 왕국 소개 랜딩페이지를 만든다.

## 영의정 판단
첫 파일럿 임무로 적합하며, 집현전-도화서-병조-사헌부 흐름을 시험하기 좋다.

## 참여 기관 배정
| 기관 | 담당 | 상태 | 산출물 |
|---|---|---|---|
| 승정원 | 작업 흐름 및 상태 기록 | 완료 | `landing/seungjeongwon-log.md` |
| 집현전 | 핵심 메시지와 구조 정의 | 완료 | `landing/jiphyeonjeon-notes.md` |
| 도화서 | 히어로 비주얼 및 시각 톤 설계 | 완료 | `landing/dohwaseo-notes.md` |
| 병조 | HTML/CSS 구현 | 완료 | `landing/byeongjo-progress.md` |
| 사헌부 | 1차 검수 및 수정권고 | 완료 | `landing/final-review.md` |

## 선행조건
- Claude CLI 사용 가능 확인
- Codex CLI 사용 가능 확인
- 파일 접근 가능 여부 확인

## 병목 보고
- 병조 단계에서 파일 잠금 의심 이슈가 발생하여 수정 흐름이 매끄럽지 않았음
- 검수 단계 종료 기준이 약해 작업 리듬이 늘어졌음
- 재시험에서는 승정원 기록과 기관별 산출 파일을 기준으로 상태 추적은 가능해졌으나, 실제 구현 마감 속도는 아직 더 개선이 필요함

## 영의정 메모
- 이 작업은 완전 재시작이 아니라 기존 1차안을 기준으로 짧게 마감해야 함
- 이후 파일럿부터 단계 분리와 상태판 운영을 강제 적용
- 이번 차수는 작업 보드 체계를 실제 적용한 첫 시험으로 간주함
- 집현전/도화서/사헌부의 산출 흔적을 실제 파일로 남겨 추적 가능성을 확보함
