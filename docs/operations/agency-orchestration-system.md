# 기관 오케스트레이션 및 모니터링 체계

## 목적
왕명이 내려졌을 때, 어떤 기관이 어떤 역할을 맡았고 현재 어떤 상태인지, 어디서 병목이 생겼는지를 영의정과 왕이 명확히 파악할 수 있도록 한다.

## 1. 필수 구성요소
- 기관 배정표
- 진행 상태판
- 병목 보고
- 기관별 산출물 기록
- 영의정 최종 취합 보고

## 2. 기관 배정표
각 작업 시작 시 아래를 기록한다.
- 작업명
- 왕명 요약
- 영의정 판단
- 참여 기관
- 기관별 담당 범위
- 선행조건

## 3. 진행 상태판
기관별 상태는 아래 중 하나로 표기한다.
- 대기
- 진행 중
- 검수 중
- 완료
- 보류
- 막힘

## 4. 병목 보고
아래 경우 병목 보고를 남긴다.
- 일정 시간 이상 진전 없음
- 환경/도구 문제
- 입력 부족
- 검수 지연
- 기관 간 인수인계 혼선

병목 보고에는 다음을 포함한다.
- 막힌 기관
- 막힌 이유
- 현재 영향
- 다음 조치

## 5. 기관별 산출물 기록
각 기관은 최소 하나의 산출 흔적을 남긴다.
- 집현전: 기획 메모
- 도화서: 시각 구조 메모
- 병조: 파일 변경 내역 및 진행 메모
- 사헌부: 검수 판정
- 승정원: 전달/상태 기록

기관 이름이 붙은 산출 파일이 실제로 남지 않으면, 해당 기관은 실무 수행을 했다고 보지 않는다.

## 6. 영의정 책임
영의정은 기관을 배정하는 데서 그치지 않고, 상태판을 갱신하고 병목을 파악하며 최종적으로 결과를 취합해 왕에게 보고한다.
