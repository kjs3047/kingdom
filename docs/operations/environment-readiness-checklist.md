# 실전 작업 전 환경 준비 체크리스트

## 목적
실제 구현, 자동화, 코딩, 외부 도구 사용에 들어가기 전에 영의정이 먼저 확인해야 할 선행조건을 정리한다.

## 공통 체크
- 필요한 도구가 설치되어 있는가
- 필요한 도구가 로그인/인증되어 있는가
- 권한 또는 정책상 문제가 없는가
- 작업 디렉터리와 파일 접근이 가능한가
- 테스트/실행이 가능한 환경인가

## CLI 중심 작업 체크
- Claude Code CLI 설치 여부
- Claude Code CLI 로그인 상태
- Codex CLI 설치 여부
- Codex CLI 로그인 상태
- 터미널 기반 작업 가능 여부
- Git 사용 가능 여부

## 운영 원칙
- 위 항목은 왕이 다시 질문하기 전에 영의정이 먼저 점검한다.
- 누락이 있으면 실전 작업 전 먼저 보고한다.
- 밴 또는 보안 위험이 있는 접근 방식은 즉시 배제한다.
