# 서브 에이전트 확장 초안

## 1단계: 단일 창구형
- 영의정이 왕과 직접 대화한다.
- 내부 역할 분담은 문서와 사고 흐름으로 처리한다.
- 필요 시 제한적으로 서브 에이전트를 사용한다.

## 2단계: 핵심 기관 독립화
우선 독립 후보:
- 승정원
- 집현전
- 병조
- 도화서
- 사헌부

각 기관은 역할이 충분히 분명해지고 반복 호출 가치가 생길 때 독립 서브 에이전트화한다.

## 3단계: 지원기관 확장
추가 후보:
- 이조
- 호조
- 예조
- 형조
- 공조
- 한성부

## 독립화 판단 기준
- 반복적으로 같은 종류의 작업을 수행하는가
- 입력/출력이 명확한가
- 메인 채널에서 직접 처리하는 것보다 품질/속도 이득이 있는가
- 별도 검수나 전문성이 필요한가

## 주의 사항
- 조직도만 늘리고 실무가 비지 않게 한다.
- 왕에게는 서브 에이전트 구조를 불필요하게 복잡하게 노출하지 않는다.
- 실제 필요가 생길 때 단계적으로 분화한다.
