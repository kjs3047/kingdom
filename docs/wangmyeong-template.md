# 왕명 템플릿

영의정과 각 기관이 왕명을 안정적으로 해석하기 위한 기본 형식이다.
왕명은 자유롭게 내려도 되나, 아래 항목이 있을수록 결과 품질이 높아진다.

## 기본 항목
- 할 일: 무엇을 원하는가
- 목적: 왜 필요한가
- 원하는 결과물: 문서, 코드, 기획안, 보고서, 이미지 구상, 발표자료 등
- 마감 또는 우선순위: 언제까지 필요한가
- 제약사항: 말투, 분량, 형식, 금지 요소
- 검수 필요 여부: 빠르게 / 보통 / 엄격하게

## 간단 예시
- 할 일: 나랏말 AI 왕국 소개문을 써라
- 목적: 외부 설명용 초안이 필요함
- 원하는 결과물: 1페이지 소개문
- 마감 또는 우선순위: 오늘 안
- 제약사항: 한국어, 너무 과장되지 않게
- 검수 필요 여부: 보통

## 짧은 왕명도 가능함
예:
- 영의정, 이 기획안을 더 깔끔한 공식 문서로 다듬어라.
- 영의정, 이 서비스의 MVP 구조를 먼저 설계하라.
- 영의정, 발표자료 초안을 만들어라.

영의정은 짧은 왕명을 받더라도 필요한 정보를 보완하여 적절한 기관에 배정한다.
