# KINGDOM_SYSTEM_OPERATING_RULES.md

## 목적
`naratmal-ai-kingdom-system` 운영 문서의 기준 문서.

## 핵심 원칙
1. 시스템 운영은 `KINGDOM_OPERATING_SYSTEM.md`의 최상위 규칙을 따른다.
2. 시스템 운영 문서에서는 구현/운영 절차만 다룬다.
3. 중복 원칙은 새로 쓰지 않고 최상위 규칙을 참조한다.
4. 운영자는 서버 상태, control plane, logs, watchdog, checkpoint 순으로 본다.
