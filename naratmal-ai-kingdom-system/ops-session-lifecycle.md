# 세션 생명주기 운영 규칙

## 목적
작업이 끝난 뒤에도 불필요한 OpenClaw 세션이 계속 남는 문제를 줄인다.

## 분류 기준
- keep: 현재 유지해야 하는 세션
- review: 영의정이 직접 확인할 세션
- candidate-cleanup: 장시간 유휴 상태로 정리 후보인 세션

## 기본 원칙
1. 메인 사용자 세션은 유지한다.
2. 최근 30분 이내 활동한 세션은 즉시 삭제 후보로 보지 않는다.
3. 120분 이상 유휴 세션은 cleanup 후보로 본다.
4. 작업 완료 후에는 반드시 `npm run ops:session-sweep`로 분류를 확인한다.
5. cleanup는 자동 삭제가 아니라 영의정 확인 후 수행한다.
