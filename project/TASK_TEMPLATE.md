# Task Folder Template

기본 경로:

```text
project/YYYY-MM-DD/task-name/
```

예:

```text
project/2026-04-01/kingdom-ops-script/
```

## 권장 구조

```text
project/YYYY-MM-DD/task-name/
  README.md
  notes.md
  outputs/
  review/
  assets/
```

## 파일 역할

### README.md
- 작업 목적
- 현재 상태
- 실행/확인 방법
- 주요 산출물 위치
- 다음 이어받기 포인트

### notes.md
- 진행 메모
- 결정 사항
- 막힌 지점
- 추후 보완 항목

### outputs/
- 최종 산출물
- 제출본
- 내보내기 파일
- 생성 결과물

### review/
- 검수 메모
- 수정 요청 사항
- 사헌부/암행어사 검토 메모

### assets/
- 작업 중 사용한 이미지, 참고 자료, 보조 파일

## 최소 시작 세트

작업 시작 시 최소한 아래 두 파일은 만드는 것을 권장한다.

```text
README.md
notes.md
```

## README.md 예시 골격

```md
# task-name

## 목적
- 이 작업에서 만들거나 정리할 것

## 현재 상태
- 시작 전 / 진행 중 / 완료

## 실행 또는 확인 방법
- 확인 절차

## 주요 파일
- outputs/... 
- assets/... 

## 다음 작업
- 바로 다음 액션
```
