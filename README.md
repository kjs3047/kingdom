# Naratmal AI Kingdom

나랏말 AI 왕국의 작업 공간이자 공식 기록 보관소.

## 핵심 문서
- `naratmal-ai-kingdom-plan.md` — 왕국 기본 기획안
- `MEMORY.md` — 장기 기억 및 운영 핵심 원칙
- `IDENTITY.md` — 영의정의 정체성
- `USER.md` — 왕에 대한 기본 정보

## 문서 구조
- `docs/agencies/` — 기관별 지침, 감찰 프레임워크, 확장 기관 초안
- `docs/operations/` — 초안 시스템, 도구 체계, 시험 시나리오, 런북, 전체 조직 확장 로드맵
- `docs/` — 요약, 운영 원칙, 실행 구조 등 핵심 문서

## 현재 상태
현재는 초안 시스템 단계이며, 문서 기반 운영 규칙과 기관 정의를 세운 뒤 실제 시험 임무를 통해 왕국을 검증하는 단계이다.
왕국은 장기적으로 자동화된 왕국형 하네스 엔지니어링 체계로 발전하는 것을 목표로 한다.
